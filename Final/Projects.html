Background changer

This program uses C# System library to access regedit and change the value of the background image url and the type. You can choose between center, mosiac, strech and tiled. Use this if you can not do this through personilization on your computer. Only effects user account. Download exe and select a file path via the sdlect file button. Once done your background will chnage. Bypasses any admin blocks and not blocked by any commerical anti-viruses.
